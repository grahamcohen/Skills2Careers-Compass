# Skills2Careers Compass

A static web prototype linking labour market intelligence to skills in demand across the East African Community.

## What it covers

- **8 countries**: Kenya, Tanzania, Uganda, Rwanda, Burundi, South Sudan, DR Congo, Somalia
- **3 sectors**: Agritech, Renewable Energy, Digital Economy
- **5 audiences**: Students/Graduates, Career Specialists, Educators, Employers, Policymakers (tailored briefings via the Compass Users drawer)

Built on the UNESCO Global Skills Tracker and the ESCO occupation taxonomy. See the in-app **About** drawer (info icon, top-left) for the full data-source list.

## Running locally

It's a static site — no build step. Open `index.html` directly in any modern browser, or serve it for proper relative-path behaviour:

```bash
# Python (works without extra installs)
python3 -m http.server 8080

# or via npm script
npm run serve
```

Then visit <http://localhost:8080>.

### Optional: linters and validators

Linter configs ship in the repo but require their npm packages:

```bash
npm install            # installs dev dependencies
npm run lint           # JS + HTML + CSS lint
npm run format         # Prettier across the codebase
npm run validate:data  # JSON schema sanity checks
npm run validate:links # HTTP-checks every link in the resource files
npm test               # smoke tests (JSON loads, key data shapes)
```

## A note on the data

This is a **working prototype**. The interface design and navigation flows are intended to be production-quality, but a portion of the data is **illustrative rather than sourced**:

- Wage figures (`wages.json`) are generated by `simulate_wages.py`
- AI Interview Coach feedback is randomised mock text
- Some outcome-scorecard percentages on training cards come from a default profile when a real one isn't available
- Featured Mentor avatars in the Community Hub (sourced from `specificMentors` in `data.js`) use `i.pravatar.cc` placeholder portraits with an initials-circle fallback when the image fails to load; an amber **Sample** pill flags the illustrative profile

Anywhere the user sees a small amber **Demo** pill or hovers a tooltip pointing to the About drawer, they're looking at this illustrative content. Course providers, training programmes, ventures, scholarships, and resource links are all real and verified.
